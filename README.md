## 주제

> 💡 새로운 지식 학습의 뼈대가 되는 Spring의 기본 원리 이해를 목표로 합니다.

- 스프링의 핵심 가치 및 원리 이해를 바탕으로 기본 개념확립
- 자바 공부를 밑바탕으로 스프링 동작원리 파악
- 코드를 통한 다양한 주제에 대한 구현
